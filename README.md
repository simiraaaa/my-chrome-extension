拡張機能を作る練習

## 開発

```
yarn
npm run dev
```

```
npm run build
```


## 機能一覧
- タブ検索、切り替え
- 前のタブへ戻る、進む
- タブのソート
- control + D で選択中(or ホバー中)のタブを閉じる
- 最初にスペースを入れてから検索でブックマーク検索
- 最初に > を入れてから検索でブックマークレット検索[^1]
- 最初に `<` を入れてから検索でリーディングリスト検索
  - 選択中(or ホバー中) control + X で未読既読をトグルする。
  - チェックアイコンクリックでも同じくトグルする。

## 実装予定の機能
- ソートの優先順位設定(optionsページ)
- ブックマーク検索、管理
- プロジェクト管理(ブックマーク検索ベース)
- よく開くタブ(URL)を初期表示で開く順に表示する (検索文字を未入力時)
- 履歴から検索
- localhost に対して post を叩く (別拡張機能で作るかも)


## ブックマークレットを使用したい場合
[^1]: chrome://extensions で詳細を開き `ユーザー スクリプトを許可する` をONにする。
